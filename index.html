About me

Hi, my name is Chris and I'm from Cologne, Germany. I'm currently doing a Bootcamp at neuefische to become a Web Developer. This is my first project to practise HTML and CSS.

Right before I started the Web Developer Bootcamp, I travelled to Sardinia with my girlfriend. We really loved this place and would highly recommend it to everyone. On this website I want to share the best tipps with you.
